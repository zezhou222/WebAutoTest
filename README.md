## Web自动化测试平台

注：python写的小项目

**目的：**
    
        为公司内部的测试人员去使用；
        平常测试人员需要安装很多工具和配置各种工具的运行环境才能进行测试，但是这个项目可以减少测试人员配置各种工具的环境的麻烦，并且降低测试人员的编码难度；
        并且所有数据由公司数据库保存，测试人员可以在内网中任何一台机器上，只需登陆就可运行测试用例及拿到测试报告。
### 项目功能：
1. 关于用例		
	- 添加用例
	- 编辑用例
	- 显示用例
	- 删除用例
	- 执行用例

2. 关于用例执行结果
	- 显示用例执行结果
	- 删除用例执行结果
	- 显示用例步骤执行的详情结果
	- 显示用例步骤执行的截图

3. 接口测试
    - 临时测试
    - 添加接口测试数据
    - 显示
    - 修改
    - 删除
    - 测试结果的显示

4. 用例测试可重用，分公共和私有用例

    公共用例可以被重复使用。
    
    使用场景：如登陆只需编写一次登陆用例，之后通过步骤选择公共用例使用即可。

5. 测试执行结果发送至用户的邮箱 

6. 可通过excel文件导入用例数据(格式是固定的)

7. 可以定时执行测试用例
    
    - 分周期、间隔、一次性任务。
